## 🎯 任务目标

### 核心目标

- **完整性**: 爬取所有215个Restek产品的详细信息
- **准确性**: 确保产品名称、型号、规格等关键信息准确无误
- **丰富性**: 尽可能获取产品描述、技术参数、应用领域、图片等
- **结构化**: 输出标准化的CSV格式，便于批量导入

### 质量标准

**必达标准** (Minimum Acceptable):
- ✅ 成功率 ≥ 90% (至少194个产品)
- ✅ 描述覆盖率 ≥ 70% (至少151个产品)
- ✅ 规格完整性 ≥ 90% (至少194个产品有≥3个规格参数)
- ✅ CSV格式正确性 100%

**优秀标准** (Excellent):
- ⭐ 成功率 ≥ 95% (至少205个产品)
- ⭐ 描述覆盖率 ≥ 80% (至少172个产品)
- ⭐ A/B级描述占比 ≥ 40% (至少86个产品的描述≥100字符)
- ⭐ 平均规格字段数 ≥ 15个
- ⭐ 图片覆盖率 ≥ 70% (至少151个产品)

## 📋 执行流程

### 阶段1：环境准备 (10分钟)

1. 安装必要的依赖包
2. 读取产品清单CSV文件
3. 初始化浏览器或HTTP客户端

### 阶段2：小批量测试 (20分钟)

1. 测试前10个产品
2. 验证URL构建逻辑
3. 检查页面结构和数据提取
4. 调整爬虫策略

### 阶段3：全量爬取 (90-120分钟)

1. 批量爬取所有215个产品
2. 实时保存中间结果
3. 处理异常和失败情况
4. 记录爬取日志

### 阶段4：质量检查 (20分钟)

1. 数据验证（必需字段、格式检查）
2. 统计质量指标（成功率、描述覆盖率等）
3. 生成质量报告
4. 分析失败原因

### 阶段5：结果提交 (10分钟)

1. 输出最终CSV文件：`restek_215_all_results.csv`
2. 生成质量报告：`RESTEK_CRAWLING_REPORT.md`
3. 提交交付文件
