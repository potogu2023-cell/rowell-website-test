# ROWELL产品数据质量改进 - 爬虫任务指令

**任务类型**: 数据补充和验证  
**优先级**: 🔴 最高  
**预计工作量**: 2,689个产品  
**目标**: 补充缺失的产品图片、规格和描述信息

---

## 📋 任务概述

当前ROWELL产品库有2,689个产品,但存在以下严重的数据质量问题:

- ❌ **100%产品缺失图片** (imageUrl)
- ❌ **100%产品缺失规格** (specifications)
- ❌ **76.9%产品缺失描述** (description)
- ⚠️ 部分产品名称被截断或不完整

本次爬虫任务的目标是从品牌官网补充这些缺失信息,提升产品库的完整性和用户体验。

---

## 🎯 爬取目标

### 必需字段 (Must Have)

1. **产品图片URL** (imageUrl)
   - 要求: 高清产品图片的直接URL
   - 格式: https://... (完整URL)
   - 优先级: 🔴 最高

2. **技术规格** (specifications)
   - 要求: 详细的技术参数和规格
   - 格式: JSON对象或结构化文本
   - 包含: 尺寸、材质、容量、技术参数等
   - 优先级: 🔴 最高

3. **产品描述** (description)
   - 要求: 完整的产品描述文本
   - 格式: 纯文本或HTML
   - 包含: 产品用途、特点、应用场景
   - 优先级: 🟡 高

### 可选字段 (Nice to Have)

4. **完整产品名称** (name)
   - 要求: 验证并更新被截断的产品名称
   - 格式: 完整文本
   - 优先级: 🟡 中

5. **产品详细描述** (detailedDescription)
   - 要求: 更详细的产品说明
   - 格式: HTML或Markdown
   - 优先级: 🟢 低

## 🔍 爬取策略建议

### 方法1: 基于零件号搜索 (推荐)

1. 使用产品的`partNumber`在品牌官网搜索
2. 找到对应的产品详情页
3. 提取所需字段
4. 验证产品名称是否匹配

**优点**: 准确率高,不易出错  
**缺点**: 需要逐个产品搜索

### 方法2: 批量API调用

如果品牌提供产品API,可以批量获取产品信息。

**优点**: 速度快,效率高  
**缺点**: 不是所有品牌都有公开API

### 方法3: 产品目录页爬取

从品牌的产品目录页批量爬取。

**优点**: 可以发现新产品  
**缺点**: 需要匹配现有产品,可能有遗漏

## 🚨 特殊注意事项

### 1. 品牌名称统一

在爬取Thermo Fisher产品时,请统一使用 **"Thermo Fisher Scientific"** 作为品牌名称。

### 2. 产品匹配

使用 `productId` 或 `brand + partNumber` 组合来匹配产品:

```
匹配规则: brand + partNumber
示例: Agilent + 5183-2068 → AGIL-51832068
```

### 3. 数据清洗

- 移除HTML标签 (如果不需要)
- 统一单位格式 (如: 2mL, 2 mL, 2ml → 2mL)
- 移除多余空格和换行符

### 4. 错误处理

如果某个产品在官网找不到:
- 在报告中标记该产品
- 说明原因 (如: 产品已停产、零件号错误等)
- 不要留空,用 "NOT_FOUND" 标记

## ⚖️ 法律合规要求

### 允许的行为 ✅

1. **访问公开产品页面**
   - 这些是公开展示的产品信息
   - 用于商业目的的产品数据库

2. **提取产品技术参数**
   - 产品规格、尺寸等事实性信息
   - 不受版权保护的数据

3. **保存产品图片URL**
   - 仅保存URL链接,不下载图片
   - 用于展示时动态加载

### 禁止的行为 ❌

1. **不要下载和存储品牌Logo**
   - 品牌Logo受商标保护
   - 仅使用产品图片

2. **不要复制营销文案**
   - 避免逐字复制营销语言
   - 可以提取技术描述

3. **不要绕过访问限制**
   - 遵守robots.txt
   - 不要使用反爬虫绕过技术

4. **不要过度请求**
   - 控制请求频率 (建议: 1-2秒/请求)
   - 避免对服务器造成负担

### 风险规避建议

1. **使用合理的User-Agent**
   - 标识为浏览器访问
   - 不要伪装成官方爬虫

2. **尊重robots.txt**
   - 检查并遵守爬取规则
   - 避免爬取禁止的路径

3. **控制并发数**
   - 单个域名: 1-2个并发
   - 避免触发反爬虫机制

4. **数据使用声明**
   - 在网站添加数据来源说明
   - 明确标注"非授权经销商"
