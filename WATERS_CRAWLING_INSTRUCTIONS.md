### CSV文件格式

**必需字段**:
```csv
productId,partNumber,brand,name,description,descriptionQuality,specifications
```

**字段说明**:
- `productId`: 产品ID（如WATS-186010613）
- `partNumber`: 零件号（如186010613）
- `brand`: 品牌名称（固定为"Waters"）
- `name`: 产品名称（从输入CSV获取）
- `description`: 产品描述（从官网爬取）
- `descriptionQuality`: 描述质量等级（high/medium/low/extracted/none）
- `specifications`: 技术规格（JSON格式字符串）

**描述质量等级标准**:
- `high`: 详细描述，≥200字符，包含产品特性和应用信息
- `medium`: 中等描述，100-200字符，包含基本产品信息
- `low`: 简短描述，50-100字符，仅包含核心信息
- `extracted`: 从产品名称或目录号提取的描述，<50字符
- `none`: 无描述

**specifications JSON格式示例**:
```json
{
  "Particle Size": "2.7 µm",
  "Column Length": "250 mm",
  "Inner Diameter": "4.6 mm",
  "Phase Type": "C18",
  "Pore Size": "100 Å",
  "pH Range": "1-12",
  "Temperature Limit": "90°C",
  "USP Classification": "L1"
}
```

## 🔧 技术要求

### 数据提取规则

**描述提取**:
1. 优先从产品详情页的"Description"或"Overview"部分提取
2. 如果没有专门的描述部分，从产品特性列表中提取
3. 清除HTML标签和多余的空白字符
4. 合并多个段落，用空格分隔
5. 限制最大长度为500字符

**规格提取**:
1. 从产品规格表（Specifications Table）中提取
2. 常见规格字段：
   - Particle Size（粒径）
   - Column Length（柱长）
   - Inner Diameter（内径）
   - Phase Type（固定相类型）
   - Pore Size（孔径）
   - pH Range（pH范围）
   - Temperature Limit（温度限制）
   - USP Classification（USP分类）
   - Surface Area（比表面积）
   - Carbon Load（碳载量）
3. 保留单位（µm, mm, Å, °C等）
4. 清除导航文本和无关信息
5. 以JSON格式存储

**质量控制**:
1. 验证每个产品的productId和partNumber匹配
2. 检查描述长度，确保不是空字符串或仅包含空白
3. 验证specifications是有效的JSON格式
4. 检查规格字段数量，确保≥3个字段
5. 记录所有错误和警告信息

### 错误处理

**常见错误**:
1. 页面不存在（404）→ 标记为失败，记录到日志
2. 页面加载超时 → 重试3次，仍失败则标记为失败
3. 描述为空 → descriptionQuality设为"none"
4. 规格表不存在 → specifications设为空对象{}

**重试策略**:
- 页面加载失败：重试3次，每次间隔5秒
- 元素未找到：等待10秒后重试，最多3次
- 网络错误：重试3次，每次间隔10秒

## ⚠️ 特殊注意事项

### Waters官网特点

1. **产品页面结构**:
   - Waters官网可能使用动态加载
   - 需要等待JavaScript渲染完成
   - 可能需要点击"Show More"按钮展开完整描述

2. **产品命名**:
   - Waters产品名称通常包含系列名、粒径、尺寸等信息
   - 示例：CORTECS Premier T3, 2.7 µm, 4.6 x 250 mm Column, 1/pk

3. **规格表格式**:
   - Waters可能使用表格或列表形式展示规格
   - 需要灵活处理不同的HTML结构

4. **产品系列**:
   - CORTECS系列：固体核心技术
   - ACQUITY UPLC系列：超高效液相色谱
   - XBridge系列：杂化粒子技术
   - Symmetry系列：经典C18色谱柱

### 可能遇到的问题

1. **动态内容加载**:
   - 解决方案：使用Selenium等待元素出现
   - 设置合理的等待时间（10-15秒）

2. **反爬虫机制**:
   - 解决方案：设置User-Agent，模拟真实浏览器
   - 控制爬取速度，避免过快请求

3. **页面结构变化**:
   - 解决方案：使用多个CSS选择器作为备选
   - 记录无法提取的页面，人工检查
