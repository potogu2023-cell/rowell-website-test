## 附录：SQL查询

### 查询所有品牌及产品数
```sql
SELECT brand, COUNT(*) as total 
FROM products 
GROUP BY brand 
ORDER BY total DESC;
```

### 查询描述缺失的产品
```sql
SELECT brand, COUNT(*) as missing_desc
FROM products 
WHERE description IS NULL OR LENGTH(description) < 100
GROUP BY brand;
```

### 查询规格缺失的产品
```sql
SELECT brand, COUNT(*) as missing_specs
FROM products 
WHERE specifications IS NULL OR specifications = ''
GROUP BY brand;
```

### 删除特定品牌（谨慎使用）
```sql
-- 示例：删除ACE品牌
DELETE FROM products WHERE brand = 'ACE';
```
